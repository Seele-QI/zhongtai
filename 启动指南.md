# AgentHub 项目启动指南

## 环境要求

| 工具      | 版本             | 检查命令                 |
| ------- | -------------- | -------------------- |
| Node.js | >= 18          | `node -v`            |
| pnpm    | 最新             | `pnpm -v`            |
| Python  | >= 3.10        | `python --version`   |
| pip 包   | uvicorn, httpx | `python -m pip list` |

## 快速启动（推荐）

### CMD 命令行

```cmd
cd /d F:\A-项目\13-中台网站
set NODE_OPTIONS=--import ./scripts/dev-fix.mjs
pnpm dev:all
```

## 访问地址

| 服务     | 地址                           |
| ------ | ---------------------------- |
| 前端     | <http://localhost:3000>      |
| 后端 API | <http://localhost:8000>      |
| API 文档 | <http://localhost:8000/docs> |

## 常用命令

```bash
pnpm dev          # 仅前端（端口 3000）
pnpm dev:all      # 前端 + 后端（3000 + 8000）
pnpm dev:api      # 仅后端（端口 8000）
pnpm build        # 生产构建
pnpm lint         # 代码检查
```
